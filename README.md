## Clone this project

```bash
> git clone https://github.com/kelvin44/Mr-Bot
```

## Install the dependencies:
Before running the below command, make sure you're in the project directory that
you've just cloned!!

```bash
> cd Mr-Bot
> bash install.sh
```

### Usage
```bash
> npm start
```

### Settings
```json
{
	"prefix": "#",
	"ownerNumber": "62858xxxxx",
	"apiKey": "Your ApiKey here"
}
```
